# COMPILATION

1. Compile all java files

   ```
   $ javac *.java
   ```

2. Compile implemented class

   ```
   $ rmic implemented class
   ```

3. Start RMI registry
   ```
   $ rmiregistry <RMI_PORT>(1029)
   ```
4. Start Server
   ```
   $ java Server
   ```
5. Start client
   ```
   $ java Client
   ```
